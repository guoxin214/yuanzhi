课程设置 > Python工程师

第一阶段

Python 入门

Python 语言已经在大数据、机器学习等领域胜出，这是毋庸置疑的事实了。它以简洁、优雅的语言风格征服了程序员和非专业程序员。并且，它还是一种非常容易学的语言。
学习知识点：基础语法、字符串和解析、界面编程、命名空间和作用域、面向对象高级语法、文件操作和数据处理等，一个实战作业。
目标：达到Python 初级程序员的水平，具备开发基础，掌握面向对象编程思想。

第二阶段

LINUX基础

Linux 是一套免费使用和自由传播的类 Unix 操作系统，在服务器占有绝对的市场份额。
要学习的基础内容：linux基础入门和架构，Linux操作系统及常用基础命令，Linux操作系统及常用命令，Linux文件管理类命令解答，bash shell全面深入技巧与实战进阶，Linux核心必备技能-用户及权限讲解。
本套课程适合零基础学习人员，是入门指导课程。

第三阶段

Git/GitHub
操作

课程背景：Git 是一款免费开源的分布式版本控制系统，用于敏捷高效地处理任何大小的项目。越来越多的项目都迁移到 Git 上进行管理，包括很多的开源项目都托管在 GitHub 上，使得 Git 成为了当下最热门的版本控制系统，因此 Git 成为了不可或缺的技能。
核心内容：Git 安装与简单配置；获取和创建仓库；git add，git commit，暂存区的理解，git rm，git mv；git branch，git merge，git tag；查看与对比历史记录；修改历史记录。
通过本课程的学习，可以使用git来管理你写的代码及进行团队协作开发，进行版本控制和管理。

第四阶段

数据库
基础入门

本课程面向数据库初学者，以实用的案例、通俗易懂的语言详细讲解 MySQL 的开发和管理技术。
知识点：初步了解数据库，为什么需要数据库，安装数据库、连接数据库、学习SQL语言基础，数据库定义，对数据进行常用的数据操作。
学习完本课程，你将掌握 MySQL 与数据库基础知识以及数据库的分析与设计方法。

第五阶段

项目级
系统实践

Django Web快速开发实践 第一个核心知识模块是Django框架本身：Django是一个开放源代码的Web应用框架，由Python写成，已成为Python语言最流行的WEB框架。Django框架学习上手简单，又自带了后台Admin，ORM ，用户认证，权限，中间件等通用组件，极大提高了WEB程序开发效率。
本课程非常细致深入的讲解了WEB的原理、django基础和进阶内容，可以快速开发Django web 系统。
第二个核心提高的知识模块：复杂数据库表，一个实际的线上项目主要难度在于表的设计与逻辑处理，是通过一对多，多对多的表关系来进行数据管理的。 学好多表的关系是设计开发复杂项目的关键。
第三个核心要提高的知识模块：学习Ajax异步发送消息，可以达到更友好的体验结果。
基于Python的O2O商城开发
本课程以目前主流的商城项目为核心，可用于实际的微店项目，通过前后端框架的结合使用，巩固 Python Web 开发相关知识点，使用上面学到的框架和多个知识模块，通过线上项目让知识融会贯通，技能在实战中得到提高。课程主要有 常用Python 框架解析、前端框架的使用、Ajax 技术讲解及商城项目的需求分析、界面设计、系统架构设计及实现。
